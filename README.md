# ZZZ Prydwen Helper

Desktop application built on C# WPF (Windows only) for **Zenless Zone Zero** that synchronizes data with <a href="https://www.prydwen.gg/zenless">**Prydwen.gg**</a> to help you build your agents efficiently.
For casuals and min-maxers, who want to quickly check their stats without browsing the site.

## Installation

1. Go to the **Releases** page on the right side of this repository.
2. Download the latest `ZZZ_Prydwen_Helper_Setup.exe` installer.
3. Relocate and run the installer. It will extract files into its execution directory (so make sure to put it in the directory where you can easly find newly created folder) and automatically place a shortcut on desktop.
  3.1 To update: Run the new installer in the same directory as the previous version not inside it. (Cote: temporary, the update system is under development)

## Roadmap

The project is currently under active development. So treat it as highly WIP state.

### Planned Features (Что ожидает реализации):
* [ ] **Comparison:** An overlay comparing your current in-game character stats side-by-side with recommended stats and build from Prydwen.
* [ ] **Best Build Showcase:** Shows you the best W-Engines and Drive Disc Sets with dynamic efficiency calculations from Prydwen.
* [ ] **Mindscapes Calculations:** Visual percentage scaling representation for Mindescapec cinema evaluated against base M0 performance (data from Prydwen) so you can see who is worth investing in obtaning an additional copies.
* [ ] **Synergies and Team Comp:** An advanced recommendation system based on your account's unlocked characters and their builds.
